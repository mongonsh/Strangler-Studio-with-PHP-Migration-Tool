HOW TO RUN
================================================================================

Terminal 1 (Backend):
  cd php-migration-tool/backend
  python -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  python main.py

Terminal 2 (Frontend):
  cd php-migration-tool/frontend
  npm install
  npm run dev

Then open: http://localhost:5173

================================================================================
WHAT IT DOES
================================================================================

1. Upload PHP project (ZIP file)
2. Analyze code (routes, models, dependencies)
3. Generate Python/FastAPI code
4. Download complete migrated project

================================================================================
FEATURES
================================================================================

✅ Beautiful web UI with 4-step wizard
✅ PHP code analysis (Laravel, Slim, Symfony, plain PHP)
✅ Route extraction (GET, POST, PUT, DELETE)
✅ Model/class extraction
✅ OpenAPI 3.0 specification generation
✅ FastAPI endpoint generation
✅ Pydantic model generation

================================================================================
API ENDPOINTS
================================================================================

POST   /api/upload              - Upload PHP project ZIP
POST   /api/analyze/{id}        - Analyze PHP code
POST   /api/generate/{id}       - Generate Python code
GET    /api/download/{id}       - Download generated project
GET    /api/preview/{id}/{file} - Preview generated file
